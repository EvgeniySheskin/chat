Приложение-чат.
Содержит два вложенных проекта - клиентское приложение и серверное. В теории - кросс-платформенные (проверялось только под Win).
Интегрирована MySQL через ODBC ANSI Driver 9.5.0. Сетевое взаимодействие возможно, настройки доступа - хардкод. 
добавлена поддержка многопоточности и код обновлено по стандарту C++17 (filesystem, lock_guard и т.п.).
